## 2. 분석 배경 및 필요성

### 현황 및 문제점

- **이탈률 증가**: 최근 6개월간 VIP 고객(연 구매액 100만원 이상) 이탈률 15% 증가
- **높은 재획득 비용**: 이탈 고객 재획득 비용이 신규 고객 획득의 3배 수준
- **낮은 재구매율**: 현재 이탈 후 대응 방식으로는 효과 미흡 (재구매율 5% 수준)
- **경쟁 심화**: 경쟁사의 공격적 프로모션 및 멤버십 제도로 고객 이동 증가

### 분석 필요성

- 이탈 징후를 사전에 포착하여 **선제적 대응** 필요
- 고객별 **맞춤형 리텐션 전략**으로 예산 효율성 제고
- LTV(고객생애가치) 높은 VIP 고객 유지를 통한 **매출 안정화**
- **데이터 기반 의사결정** 체계 구축

### 선행 분석 이력

- **2023년 Q2**: 전체 고객 세그멘테이션 분석 (RFM 기반)
- **2023년 Q4**: 고객 만족도 조사 분석
- **비고**: 예측 모델은 최초 구축

## 3. 분석 목표

### 비즈니스 목표

VIP 고객 이탈률을 현재 15%에서 **10% 이하로 감소**시켜 연간 **30억원의 매출 손실 방지**

### 분석 목표 (SMART 원칙)

- **Specific (구체적):** VIP 고객의 향후 3개월 이내 이탈 확률 예측 모델 구축
- **Measurable (측정가능):** 예측 정확도(F1-Score) 0.75 이상 달성
- **Achievable (달성가능):** 2년치 고객 거래 데이터 및 행동 로그 확보
- **Relevant (관련성):** 마케팅 비용 30% 절감 및 ROI 200% 향상
- **Time-bound (기한):** 3개월 내 모델 개발 완료 및 운영 시스템 연계

### 성공 기준 (KPI)

| 지표명 | 현재 값 | 목표 값 | 측정 방법 |
|--------|---------|---------|-----------|
| VIP 이탈률 | 15% | 10% | 월별 이탈 고객 수 |
| 예측 정확도(F1) | - | 0.75 | 테스트셋 평가 |
| 리텐션 캠페인 ROI | 150% | 200% | (수익-비용)/비용 |
| 조기 감지율 | - | 70% | 이탈 1개월 전 포착 |

## 4. 분석 범위

### 포함 범위 (In-Scope)

- **분석 대상:** 연 구매액 100만원 이상 VIP 고객 약 50,000명
- **분석 기간:** 2022-01-01 ~ 2023-12-31 (2년)
- **분석 채널:** 온라인몰, 모바일앱 (오프라인 제외)
- **분석 지표:**
  - 구매 행동: RFM, 객단가, 구매 카테고리, 할인 민감도
  - 서비스 이용: 로그인 빈도, 체류시간, 찜/장바구니
  - 고객 응대: CS 문의, 반품/교환, 만족도 평가
  - 마케팅 반응: 이메일/SMS 오픈율, 프로모션 참여

### 제외 범위 (Out-of-Scope)

- 일반 고객(연 구매액 100만원 미만) 이탈 분석
- 오프라인 매장 구매 데이터
- 외부 데이터(SNS, 경쟁사 정보 등)
- B2B 법인 고객

## 6. 분석 방법론

### 분석 접근법

- [x] 탐색적 분석 (EDA)
- [x] 통계 분석
- [x] 머신러닝/딥러닝
- [ ] 시뮬레이션

### 예상 분석 기법

#### Phase 1: 탐색적 분석
- **코호트 분석**: 가입 시점별 이탈 패턴
- **RFM 세그멘테이션**: 고객 그룹별 특성 파악
- **생존 분석**(Survival Analysis): 이탈 시점 분포

#### Phase 2: 특징 엔지니어링
- **구매 패턴 지표**: 구매 간격 변화율, 구매 빈도 추세
- **행동 변화 지표**: 로그인 감소율, 체류시간 하락
- **고객 가치 지표**: CLV, RFM 스코어 변화
- **인터랙션 지표**: CS 문의 증가, 부정 리뷰

#### Phase 3: 예측 모델링
- **알고리즘 후보**:
  - Logistic Regression (baseline)
  - Random Forest
  - XGBoost / LightGBM
  - Neural Network (Dense)
- **평가 지표**: F1-Score, AUC-ROC, Precision@K
- **검증 방법**: 시계열 교차검증 (Time-series CV)

#### Phase 4: 해석 및 전략 수립
- **SHAP values**로 이탈 요인 해석
- 고객 세그먼트별 맞춤 전략 도출

### 분석 도구

- Python 3.9 (pandas, scikit-learn, XGBoost, SHAP)
- SQL (BigQuery)
- Tableau (시각화)
- Jupyter Notebook / VS Code
