## 2. 분석 배경 및 필요성

### 현황 및 문제점

- **높은 불량률**: A 라인 불량률 현재 2.5% (목표 1.0%)
- **늦은 발견**: 불량 발견 시점이 후공정 검수 단계 (생산 후 2시간)
- **경제적 손실**: 연간 불량 손실액 약 **50억원**
- **고객 불만**: 사후 대응 방식으로 리드타임 지연 및 고객 클레임 발생

### 분석 필요성

- 불량 발생 전 **조기 감지**로 사전 조치 필요
- 생산 파라미터 최적화를 통한 **수율 향상**
- **스마트팩토리 전환**의 핵심 과제
- 글로벌 경쟁력 확보를 위한 **품질 혁신**

### 선행 분석 이력

**없음** (최초 데이터 분석 과제)

## 3. 분석 목표

### 비즈니스 목표

불량률을 현재 2.5%에서 **1.0% 이하로 감소**시켜 연간 **30억원의 품질 비용 절감**

### 분석 목표 (SMART 원칙)

- **Specific (구체적):** A 생산라인의 불량 발생 10분 전 예측 모델 구축
- **Measurable (측정가능):** 예측 정확도(Recall) 90% 이상 달성
- **Achievable (달성가능):** 6개월치 센서 데이터 및 품질 검사 결과 확보
- **Relevant (관련성):** 불량률 1.5%p 감소로 연 30억원 절감
- **Time-bound (기한):** 1개월 내 POC 완료, 3개월 내 전체 라인 적용

### 성공 기준 (KPI)

| 지표명 | 현재 값 | 목표 값 | 측정 방법 |
|--------|---------|---------|-----------|
| 불량률 | 2.5% | 1.0% | 일별 불량 개수/생산량 |
| 예측 정확도(Recall) | - | 90% | 실제 불량 포착률 |
| 조기 경보 리드타임 | 2시간 | 10분 | 불량 예측~발생 시간 |
| 품질 비용 절감액 | - | 30억 | 연간 누적 절감액 |

## 4. 분석 범위

### 포함 범위 (In-Scope)

- **분석 대상:** A 생산라인 (사출 공정)
- **분석 기간:** 2023-08-01 ~ 2024-01-31 (6개월)
- **분석 대상 불량:** 치수 불량, 외관 불량, 강도 부족
- **입력 변수:**
  - 공정 파라미터: 온도, 압력, 속도 (20개 센서)
  - 원자재 정보: LOT 번호, 수분율, 점도
  - 환경 정보: 작업장 온습도, 시간대
  - 설비 정보: 가동 시간, 유지보수 이력

### 제외 범위 (Out-of-Scope)

- B, C 라인 (차후 확대 적용 예정)
- 포장/물류 공정
- 원자재 입고 품질 검사

## 6. 분석 방법론

### 분석 접근법

- [x] 탐색적 분석 (EDA)
- [x] 통계 분석
- [x] 머신러닝/딥러닝
- [ ] 시뮬레이션

### 예상 분석 기법

#### Phase 1: 데이터 전처리
- **이상치 탐지 및 제거**: IQR, Isolation Forest
- **결측치 보간**: 선형, 시계열 기반
- **센서 신호 정규화**

#### Phase 2: 특징 공학
- **시계열 특징**: 이동평균, 표준편차, 변화율
- **도메인 특징**: 온도-압력 상호작용, 사이클 타임
- **집계 특징**: 최근 1시간/30분/10분 통계량

#### Phase 3: 모델링
**Time-series Classification:**
- LSTM (Long Short-Term Memory)
- 1D CNN
- Transformer

**Tabular Classification:**
- XGBoost
- LightGBM

**Ensemble**

#### Phase 4: 해석
- **LIME/SHAP**으로 주요 불량 원인 파악
- 파라미터 최적 범위 도출

### 분석 도구

- Python (TensorFlow, PyTorch, scikit-learn)
- Apache Kafka (실시간 스트리밍)
- MLflow (모델 관리)
- Grafana (모니터링 대시보드)
