# 다층 신경망(Multi-Layer Neural Network) 종합 이론 정리

## 1. 다층 신경망 개요

### 1.1 기본 구조

```
입력층 → 은닉층₁ → 은닉층₂ → ... → 은닉층ₙ → 출력층
```

- **입력층(Input Layer)**: 외부 데이터를 받아들이는 층
- **은닉층(Hidden Layer)**: 입력과 출력 사이에서 특징을 학습하는 층
- **출력층(Output Layer)**: 최종 예측값을 출력하는 층

### 1.2 심층 신경망(Deep Neural Network)

은닉층이 2개 이상인 신경망을 **심층 신경망(DNN)**이라 하며, 이를 학습시키는 것을 **딥러닝(Deep Learning)**이라 함.

---

## 2. 은닉층 수에 따른 문제점

### 2.1 은닉층이 너무 적을 때

| 문제 | 설명 |
|------|------|
| **과소적합(Underfitting)** | 모델의 복잡도가 낮아 데이터의 패턴을 충분히 학습하지 못함 |
| **표현력 부족** | 비선형적이고 복잡한 관계를 표현할 수 없음 |
| **높은 편향(High Bias)** | 학습 데이터에서도 성능이 낮음 |
| **특징 추출 한계** | 고수준의 추상적 특징을 학습하기 어려움 |

**증상**
- 훈련 오차와 검증 오차 모두 높음
- 학습 곡선이 빠르게 수렴하지만 낮은 성능에서 정체

### 2.2 은닉층이 너무 많을 때

| 문제 | 설명 |
|------|------|
| **과적합(Overfitting)** | 훈련 데이터에 과도하게 맞춰져 일반화 성능 저하 |
| **기울기 소실(Vanishing Gradient)** | 역전파 시 기울기가 0에 가까워져 학습 불가 |
| **기울기 폭발(Exploding Gradient)** | 기울기가 기하급수적으로 커져 학습 불안정 |
| **계산 비용 증가** | 파라미터 수 증가로 메모리와 연산 시간 급증 |
| **높은 분산(High Variance)** | 훈련 데이터에는 좋지만 새로운 데이터에 민감 |

**증상**
- 훈련 오차는 낮지만 검증 오차는 높음
- 학습이 매우 느리거나 진행되지 않음 (기울기 문제)

---

## 3. 기울기 소실 문제 (Vanishing Gradient Problem)

### 3.1 정의

역전파(Backpropagation) 과정에서 **출력층에서 입력층으로 갈수록 기울기가 점점 작아져** 초기 층의 가중치가 거의 업데이트되지 않는 현상

### 3.2 발생 원인

#### 수학적 원인

역전파에서 기울기는 **연쇄 법칙(Chain Rule)**에 의해 계산됨:

$$\frac{\partial L}{\partial W_1} = \frac{\partial L}{\partial a_n} \cdot \frac{\partial a_n}{\partial a_{n-1}} \cdot ... \cdot \frac{\partial a_2}{\partial a_1} \cdot \frac{\partial a_1}{\partial W_1}$$

각 층의 미분값이 **1보다 작은 값이 연속으로 곱해지면** 기울기가 기하급수적으로 감소

#### 활성화 함수의 영향

| 활성화 함수 | 미분값 범위 | 기울기 소실 위험 |
|-------------|-------------|------------------|
| **Sigmoid** | (0, 0.25] | **매우 높음** |
| **Tanh** | (0, 1] | **높음** |
| **ReLU** | {0, 1} | 낮음 (Dead ReLU 문제 존재) |
| **Leaky ReLU** | {α, 1} (α≈0.01) | 매우 낮음 |

### 3.3 기울기 소실 발생 조건

```
✓ 깊은 신경망 (은닉층 많음)
✓ Sigmoid 또는 Tanh 활성화 함수 사용
✓ 부적절한 가중치 초기화
✓ 학습률이 너무 낮음
```

### 3.4 Sigmoid 함수의 기울기 소실 예시

Sigmoid 함수: $\sigma(x) = \frac{1}{1+e^{-x}}$

미분: $\sigma'(x) = \sigma(x)(1-\sigma(x))$

**최대 미분값 = 0.25** (x=0일 때)

10개 층을 통과하면: $(0.25)^{10} = 0.00000095$ → 거의 0

## 4. 기울기 폭발 문제 (Exploding Gradient Problem)

### 4.1 정의

역전파 과정에서 **기울기가 기하급수적으로 증가**하여 가중치 업데이트가 불안정해지는 현상

### 4.2 발생 원인

- 각 층의 미분값이 **1보다 큰 값이 연속으로 곱해짐**
- 부적절한 가중치 초기화 (너무 큰 초기값)
- 특히 **RNN(순환 신경망)**에서 자주 발생

### 4.3 증상

- 손실 함수 값이 NaN 또는 Inf로 발산
- 가중치 값이 급격히 증가
- 학습이 불안정하고 진동

---

## 5. 기울기 문제 해결 방법

### 5.1 활성화 함수 개선

| 활성화 함수 | 수식 | 장점 | 단점 |
|-------------|------|------|------|
| **ReLU** | $max(0, x)$ | 기울기 소실 완화, 계산 빠름 | Dead ReLU 문제 |
| **Leaky ReLU** | $max(αx, x)$ | Dead ReLU 해결 | α 하이퍼파라미터 |
| **PReLU** | $max(αx, x)$ (α 학습) | α를 학습으로 최적화 | 파라미터 증가 |
| **ELU** | $x$ if $x>0$, $α(e^x-1)$ otherwise | 음수 출력 가능, 평균 0 근접 | 지수 연산 비용 |
| **SELU** | λ·ELU(x) | 자기 정규화 효과 | 특정 조건 필요 |
| **Swish** | $x \cdot \sigma(x)$ | 깊은 네트워크에서 우수 | 계산 비용 |
| **GELU** | $x \cdot Φ(x)$ | Transformer에서 우수 | 계산 비용 |

### 5.2 가중치 초기화 방법

| 초기화 방법 | 적용 활성화 함수 | 분산 공식 |
|-------------|------------------|-----------|
| **Xavier (Glorot)** | Sigmoid, Tanh | $Var(W) = \frac{2}{n_{in} + n_{out}}$ |
| **He (Kaiming)** | ReLU 계열 | $Var(W) = \frac{2}{n_{in}}$ |
| **LeCun** | SELU | $Var(W) = \frac{1}{n_{in}}$ |

**핵심 원리**: 각 층의 출력 분산을 일정하게 유지하여 기울기 흐름 안정화

### 5.3 배치 정규화 (Batch Normalization)

```
입력 → 정규화 → 스케일/시프트 → 활성화 함수
```

**동작 원리**:
1. 미니배치 내 평균과 분산 계산
2. 정규화: $\hat{x} = \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}}$
3. 스케일/시프트: $y = \gamma \hat{x} + \beta$ (학습 가능한 파라미터)

| 장점 | 설명 |
|------|------|
| 내부 공변량 변화 감소 | 각 층 입력 분포 안정화 |
| 높은 학습률 사용 가능 | 학습 속도 향상 |
| 정규화 효과 | 과적합 방지 (약한 Dropout 효과) |
| 기울기 흐름 개선 | 기울기 소실/폭발 완화 |

### 5.4 잔차 연결 (Residual Connection / Skip Connection)

```
        ┌─────────────────────┐
        │                     │
x ──────┼───→ [층] ───→ (+) ──┼───→ 출력
        │       F(x)    ↑     │
        │               │     │
        └───────────────┘     
             항등 연결 (x)
```

**출력**: $H(x) = F(x) + x$

| 효과 | 설명 |
|------|------|
| 기울기 고속도로 | 기울기가 덧셈을 통해 직접 전파 |
| 항등 매핑 학습 용이 | F(x)=0만 학습하면 됨 |
| 매우 깊은 네트워크 가능 | ResNet-152, ResNet-1000 등 |

### 5.5 기울기 클리핑 (Gradient Clipping)

기울기 폭발 방지를 위해 기울기 크기를 제한

**방법 1: 값 클리핑**
```python
if gradient > threshold:
    gradient = threshold
elif gradient < -threshold:
    gradient = -threshold
```

**방법 2: 노름 클리핑**
```python
if ||gradient|| > max_norm:
    gradient = gradient × (max_norm / ||gradient||)
```

### 5.6 기타 해결 방법

| 방법 | 설명 | 효과 |
|------|------|------|
| **LSTM/GRU** | 게이트 메커니즘으로 장기 의존성 학습 | RNN의 기울기 소실 해결 |
| **Layer Normalization** | 배치 대신 층 단위 정규화 | 배치 크기 독립적 |
| **학습률 스케줄링** | 학습률을 점진적으로 감소 | 학습 안정화 |
| **적응적 옵티마이저** | Adam, RMSprop 등 | 파라미터별 학습률 조정 |

## 6. 과적합 vs 과소적합 종합 비교

| 구분 | 과소적합 (Underfitting) | 과적합 (Overfitting) |
|------|------------------------|---------------------|
| **원인** | 모델 복잡도 부족 | 모델 복잡도 과다 |
| **은닉층** | 너무 적음 | 너무 많음 |
| **편향-분산** | 높은 편향, 낮은 분산 | 낮은 편향, 높은 분산 |
| **훈련 오차** | 높음 | 낮음 |
| **검증 오차** | 높음 | 높음 |
| **학습 곡선** | 수렴하지만 높은 오차 | 훈련/검증 격차 큼 |
| **해결책** | 층/뉴런 추가, 더 복잡한 모델 | 정규화, 드롭아웃, 조기종료 |

---

## 7. 기울기 소실 vs 기울기 폭발 비교

| 구분 | 기울기 소실 (Vanishing) | 기울기 폭발 (Exploding) |
|------|------------------------|------------------------|
| **현상** | 기울기 → 0 수렴 | 기울기 → ∞ 발산 |
| **원인** | 미분값 < 1 연속 곱셈 | 미분값 > 1 연속 곱셈 |
| **영향 층** | 입력층에 가까운 층 | 전체 네트워크 |
| **증상** | 학습 정체, 초기 층 미학습 | NaN/Inf, 발산, 불안정 |
| **주요 원인 함수** | Sigmoid, Tanh | - |
| **해결 활성화 함수** | ReLU 계열 | - |
| **해결 초기화** | Xavier, He | He |
| **특수 해결법** | 잔차 연결, LSTM | 기울기 클리핑 |

## 8. 활성화 함수 종합 비교표

| 함수 | 수식 | 출력 범위 | 미분 범위 | 기울기 소실 | 계산 비용 | 주요 용도 |
|------|------|-----------|-----------|-------------|-----------|-----------|
| **Sigmoid** | $\frac{1}{1+e^{-x}}$ | (0, 1) | (0, 0.25] | 심각 | 높음 | 이진 분류 출력층 |
| **Tanh** | $\frac{e^x-e^{-x}}{e^x+e^{-x}}$ | (-1, 1) | (0, 1] | 있음 | 높음 | RNN, 은닉층 |
| **ReLU** | $max(0, x)$ | [0, ∞) | {0, 1} | 완화 | 낮음 | CNN, DNN 은닉층 |
| **Leaky ReLU** | $max(αx, x)$ | (-∞, ∞) | {α, 1} | 완화 | 낮음 | 범용 은닉층 |
| **ELU** | 조건부 | (-α, ∞) | 연속 | 완화 | 중간 | 깊은 네트워크 |
| **Softmax** | $\frac{e^{x_i}}{\sum e^{x_j}}$ | (0, 1) | - | - | 중간 | 다중 분류 출력층 |

## 9. 가중치 초기화 방법 비교표

| 초기화 | 분포 | 분산 | 적합한 활성화 함수 | 특징 |
|--------|------|------|-------------------|------|
| **Zero** | 상수 0 | 0 | 없음 | 대칭성 문제, 사용 금지 |
| **Random** | 균등/정규 | 고정 | - | 층 깊어지면 불안정 |
| **Xavier Uniform** | 균등 | $\frac{6}{n_{in}+n_{out}}$ | Sigmoid, Tanh | 대칭 활성화 함수용 |
| **Xavier Normal** | 정규 | $\frac{2}{n_{in}+n_{out}}$ | Sigmoid, Tanh | 대칭 활성화 함수용 |
| **He Uniform** | 균등 | $\frac{6}{n_{in}}$ | ReLU 계열 | 비대칭 활성화 함수용 |
| **He Normal** | 정규 | $\frac{2}{n_{in}}$ | ReLU 계열 | 비대칭 활성화 함수용 |

## 10. 정규화 기법 비교표

| 기법 | 정규화 축 | 배치 의존성 | 주요 적용 | 장점 |
|------|-----------|-------------|-----------|------|
| **Batch Norm** | 배치 | 있음 | CNN, DNN | 학습 속도 향상, 정규화 효과 |
| **Layer Norm** | 층 | 없음 | RNN, Transformer | 작은 배치, 가변 길이 |
| **Instance Norm** | 채널별 | 없음 | Style Transfer | 스타일 독립적 정규화 |
| **Group Norm** | 그룹 | 없음 | 작은 배치 | 배치 크기 무관 |

## 11. 최적화 알고리즘과 기울기 문제

| 옵티마이저 | 특징 | 기울기 문제 대응 |
|------------|------|------------------|
| **SGD** | 기본 경사하강법 | 대응 없음 |
| **Momentum** | 관성 추가 | 지역 최소 탈출 |
| **RMSprop** | 기울기 제곱 이동평균 | 학습률 자동 조정 |
| **Adam** | Momentum + RMSprop | 적응적 학습률, 안정적 |
| **AdamW** | Adam + 가중치 감쇠 | 정규화 효과 추가 |

## 12. 실전 가이드라인

### 12.1 은닉층 수 결정

```
1. 단순 문제 → 1-2개 은닉층
2. 중간 복잡도 → 3-5개 은닉층
3. 복잡한 문제 (이미지, 텍스트) → 10개 이상 + 잔차 연결
```

### 12.2 기울기 문제 예방 체크리스트

```
□ ReLU 또는 변형 활성화 함수 사용
□ He 초기화 (ReLU) 또는 Xavier 초기화 (Tanh)
□ 배치 정규화 적용
□ 깊은 네트워크는 잔차 연결 사용
□ 적절한 학습률 설정
□ 적응적 옵티마이저 (Adam) 사용
□ 기울기 클리핑 (RNN의 경우)
```

### 12.3 권장 조합

| 네트워크 유형 | 활성화 함수 | 초기화 | 정규화 | 옵티마이저 |
|---------------|-------------|--------|--------|------------|
| **일반 DNN** | ReLU | He | Batch Norm | Adam |
| **CNN** | ReLU | He | Batch Norm | SGD+Momentum |
| **RNN** | Tanh | Xavier | Layer Norm | Adam |
| **Transformer** | GELU | Xavier | Layer Norm | AdamW |
| **GAN** | LeakyReLU | He | Spectral Norm | Adam |

## 13. 핵심 요약 표

| 문제 | 발생 조건 | 증상 | 해결 방법 |
|------|-----------|------|-----------|
| **과소적합** | 은닉층 부족, 모델 단순 | 훈련/검증 오차 모두 높음 | 층/뉴런 추가 |
| **과적합** | 은닉층 과다, 데이터 부족 | 훈련↓ 검증↑ 격차 | 정규화, 드롭아웃, 조기종료 |
| **기울기 소실** | Sigmoid/Tanh, 깊은 층 | 학습 정체, 초기 층 미학습 | ReLU, He초기화, 잔차연결 |
| **기울기 폭발** | 큰 가중치, RNN | NaN, 발산 | 기울기 클리핑, 적절한 초기화 |
