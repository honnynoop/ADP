# PART 1. t검정 (t-Test)

## 1-1. 단일 표본 t검정 (One-Sample t-Test)

### 개념
모분산 **σ² 미지** 상태에서 **하나의 표본 평균**이 특정 값과 같은지 검정

$$t = \frac{\bar{X} - \mu_0}{S / \sqrt{n}} \sim t(n-1)$$

### 문제
> 어느 회사 직원들의 평균 출근 시간이 **30분**이라고 알려져 있다.  
> **n = 10명** 을 랜덤 샘플링한 결과:  
> **데이터 (분): 28, 32, 27, 35, 30, 29, 33, 31, 26, 34**  
> α = 0.05, 양측 검정으로 평균이 30분인지 검정하라.

### 풀이

**① 기초 통계 계산**

$$\sum x_i = 28+32+27+35+30+29+33+31+26+34 = 305$$

$$\bar{X} = \frac{305}{10} = 30.5$$

$$\sum x_i^2 = 784+1024+729+1225+900+841+1089+961+676+1156 = 9385$$

$$S^2 = \frac{\sum x_i^2 - n\bar{X}^2}{n-1} = \frac{9385 - 10 \times 30.5^2}{9} = \frac{9385 - 9302.5}{9} = \frac{82.5}{9} = 9.167$$

$$S = \sqrt{9.167} = 3.028$$

**② 가설 설정**
$$H_0: \mu = 30 \qquad H_1: \mu \neq 30$$

**③ t 통계량 계산**
$$t = \frac{\bar{X} - \mu_0}{S/\sqrt{n}} = \frac{30.5 - 30}{3.028/\sqrt{10}} = \frac{0.5}{3.028/3.162} = \frac{0.5}{0.9574} = 0.522$$

**④ 기각역 확인**

자유도 $\nu = n - 1 = 9$, 양측 $\alpha = 0.05$:

$$t_{0.025,\,9} = 2.262$$

$$|t| = 0.522 < 2.262 \implies H_0 \text{ 채택}$$

**⑤ p-값**
$$p\text{-value} \approx 0.614 > 0.05 \implies H_0 \text{ 채택}$$

**⑥ 95% 신뢰구간**
$$30.5 \pm 2.262 \times 0.9574 = 30.5 \pm 2.166 = (28.334,\; 32.666)$$

**→ 결론:** 유의수준 5%에서 평균 출근 시간이 30분이라는 귀무가설을 **기각할 수 없다** (t = 0.522, p = 0.614).

## 1-2. 독립 두 표본 t검정 (Independent Two-Sample t-Test)

### 개념
두 독립 집단의 평균 차이를 검정 (모분산 미지, 등분산 가정)

$$t = \frac{(\bar{X}_1 - \bar{X}_2)}{S_p \sqrt{\dfrac{1}{n_1} + \dfrac{1}{n_2}}} \sim t(n_1+n_2-2)$$

**합동 분산 (Pooled Variance):**

$$S_p^2 = \frac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1+n_2-2}$$

### 문제
> 남학생(그룹1)과 여학생(그룹2)의 수학 점수를 비교한다.  
> **그룹1 (남): 85, 78, 92, 88, 76** (n₁=5)  
> **그룹2 (여): 90, 85, 93, 87, 82, 91** (n₂=6)  
> α = 0.05, 양측 검정

### 풀이

**① 기초 통계**

| | 그룹1 (남) | 그룹2 (여) |
|--|--|--|
| n | 5 | 6 |
| $\sum x$ | 419 | 528 |
| $\bar{X}$ | **83.8** | **88.0** |
| $\sum x^2$ | 35213 | 46548 |
| $S^2$ | $\frac{35213-5\times83.8^2}{4} = \frac{35213-35108.2}{4} = \frac{104.8}{4} = \mathbf{26.20}$ | $\frac{46548-6\times88^2}{5} = \frac{46548-46464}{5} = \frac{84}{5} = \mathbf{16.80}$ |
| $S$ | 5.119 | 4.099 |

**② 가설 설정**
$$H_0: \mu_1 - \mu_2 = 0 \qquad H_1: \mu_1 - \mu_2 \neq 0$$

**③ 합동 분산 계산**
$$S_p^2 = \frac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1+n_2-2} = \frac{4 \times 26.20 + 5 \times 16.80}{9} = \frac{104.8 + 84.0}{9} = \frac{188.8}{9} = 20.978$$

$$S_p = \sqrt{20.978} = 4.580$$

**④ t 통계량 계산**
$$t = \frac{\bar{X}_1 - \bar{X}_2}{S_p\sqrt{\dfrac{1}{n_1}+\dfrac{1}{n_2}}} = \frac{83.8 - 88.0}{4.580\sqrt{\dfrac{1}{5}+\dfrac{1}{6}}} = \frac{-4.2}{4.580 \times 0.6055} = \frac{-4.2}{2.773} = -1.515$$

**⑤ 기각역 확인**

자유도 $\nu = 5 + 6 - 2 = 9$, $t_{0.025,9} = 2.262$:

$$|t| = 1.515 < 2.262 \implies H_0 \text{ 채택}$$

**⑥ p-값**
$$p\text{-value} \approx 0.164 > 0.05 \implies H_0 \text{ 채택}$$

**→ 결론:** 유의수준 5%에서 남녀 수학 점수 평균 차이는 **통계적으로 유의하지 않다** (t = -1.515, p = 0.164).

## 1-3. 대응 표본 t검정 (Paired t-Test)

### 개념
동일 대상에 대해 처리 전후 차이를 검정. 차이 $D_i = X_{i,\text{후}} - X_{i,\text{전}}$ 를 이용:

$$t = \frac{\bar{D}}{S_D/\sqrt{n}} \sim t(n-1)$$

### 문제
> 다이어트 프로그램 전후 체중(kg) 변화 (n=7):

| 대상 | 전(kg) | 후(kg) | D = 후-전 |
|------|--------|--------|-----------|
| 1 | 80 | 76 | -4 |
| 2 | 72 | 70 | -2 |
| 3 | 85 | 80 | -5 |
| 4 | 90 | 84 | -6 |
| 5 | 75 | 73 | -2 |
| 6 | 88 | 83 | -5 |
| 7 | 78 | 76 | -2 |

α = 0.05, 단측 검정 (프로그램이 체중 감소에 효과적인지)

### 풀이

**① D 기초 통계**

$$\bar{D} = \frac{-4-2-5-6-2-5-2}{7} = \frac{-26}{7} = -3.714$$

$$S_D^2 = \frac{\sum D_i^2 - n\bar{D}^2}{n-1} = \frac{(16+4+25+36+4+25+4) - 7 \times 3.714^2}{6} = \frac{114 - 96.60}{6} = \frac{17.40}{6} = 2.900$$

$$S_D = 1.703$$

**② 가설 설정**
$$H_0: \mu_D \geq 0 \qquad H_1: \mu_D < 0 \quad \text{(체중 감소)}$$

**③ t 통계량**
$$t = \frac{\bar{D}}{S_D/\sqrt{n}} = \frac{-3.714}{1.703/\sqrt{7}} = \frac{-3.714}{1.703/2.646} = \frac{-3.714}{0.6436} = -5.772$$

**④ 기각역**

자유도 $\nu = 6$, 좌측 단측 $t_{0.05,6} = -1.943$:

$$t = -5.772 < -1.943 \implies H_0 \text{ 기각}$$

**→ 결론:** 유의수준 5%에서 다이어트 프로그램은 체중 감소에 **유의한 효과가 있다** (t = -5.772, p < 0.001).

# PART 2. F검정 (F-Test) — 등분산 검정

## 2-1. 개념

두 모집단의 **분산이 같은지** 검정 (Levene's test 전신)

$$F = \frac{S_1^2}{S_2^2} \sim F(n_1-1,\; n_2-1)$$

> 관례: 큰 분산을 분자에 놓아 $F \geq 1$ 이 되게 함

### 문제
> A 기계(n₁=11)와 B 기계(n₂=9)로 생산한 부품 길이(mm)의 분산을 비교한다.  
> 표본 분산: **S₁² = 8.4**, **S₂² = 3.6**  
> α = 0.10, 양측 검정으로 두 기계의 분산이 같은지 검정하라.

### 풀이

**① 가설 설정**
$$H_0: \sigma_1^2 = \sigma_2^2 \qquad H_1: \sigma_1^2 \neq \sigma_2^2$$

**② F 통계량** (큰 분산을 분자)
$$F = \frac{S_1^2}{S_2^2} = \frac{8.4}{3.6} = 2.333$$

**③ 자유도**
$$\nu_1 = n_1 - 1 = 10, \qquad \nu_2 = n_2 - 1 = 8$$

**④ 기각역 확인**

양측 $\alpha = 0.10$ → 단측 상한 $\alpha/2 = 0.05$:

$$F_{0.05,\;10,\;8} = 3.347$$

$$F = 2.333 < 3.347 \implies H_0 \text{ 채택}$$

**⑤ p-값**
$$p\text{-value} \approx 0.224 > 0.10 \implies H_0 \text{ 채택}$$

**→ 결론:** 유의수준 10%에서 두 기계의 분산은 **유의한 차이가 없다** (F = 2.333, p = 0.224).  
→ 따라서 두 모집단의 등분산 가정 충족 → **합동 분산을 사용하는 t검정** 적용 가능.

# PART 3. 일원배치 ANOVA (One-Way ANOVA)

## 3-1. 개념

**3개 이상** 집단의 평균이 모두 같은지를 **분산 분해**로 검정

$$H_0: \mu_1 = \mu_2 = \cdots = \mu_k \qquad H_1: \text{적어도 하나의 } \mu_i \neq \mu_j$$

### ANOVA 분산 분해

$$SS_T = SS_B + SS_W$$

| 항목 | 이름 | 수식 | 자유도 |
|------|------|------|--------|
| $SS_T$ | 총 제곱합 | $\sum\sum(x_{ij} - \bar{X})^2$ | $N-1$ |
| $SS_B$ | 집단 간 제곱합 (Between) | $\sum_i n_i(\bar{X}_i - \bar{X})^2$ | $k-1$ |
| $SS_W$ | 집단 내 제곱합 (Within) | $\sum\sum(x_{ij} - \bar{X}_i)^2$ | $N-k$ |

**F 통계량:**

$$\boxed{F = \frac{MS_B}{MS_W} = \frac{SS_B/(k-1)}{SS_W/(N-k)} \sim F(k-1,\; N-k)}$$

---

## 3-2. 수치 예시

### 문제
> 비료 종류 A, B, C 에 따른 작물 수확량(kg)을 비교한다.

| 비료 A | 비료 B | 비료 C |
|--------|--------|--------|
| 20 | 30 | 25 |
| 22 | 28 | 27 |
| 19 | 32 | 24 |
| 21 | 29 | 26 |
| 23 | 31 | 23 |

α = 0.05, 세 비료의 평균 수확량이 같은지 검정하라.

---

### 풀이

**① 기초 통계 계산**

| 통계량 | 비료 A | 비료 B | 비료 C | 전체 |
|--------|--------|--------|--------|------|
| $n_i$ | 5 | 5 | 5 | N=15 |
| $\sum x_{ij}$ | 105 | 150 | 125 | 380 |
| $\bar{X}_i$ | **21.0** | **30.0** | **25.0** | $\bar{X}$=**25.333** |
| $\sum x_{ij}^2$ | 2215 | 4510 | 3135 | 9860 |

전체 평균:
$$\bar{X} = \frac{105+150+125}{15} = \frac{380}{15} = 25.333$$

---

**② SS_B (집단 간 제곱합) 계산**

$$SS_B = \sum_{i=1}^{k} n_i (\bar{X}_i - \bar{X})^2$$

$$= 5(21.0-25.333)^2 + 5(30.0-25.333)^2 + 5(25.0-25.333)^2$$

$$= 5 \times 18.775 + 5 \times 21.781 + 5 \times 0.111$$

$$= 93.875 + 108.905 + 0.555 = \mathbf{203.333}$$

자유도: $df_B = k - 1 = 3 - 1 = 2$

$$MS_B = \frac{203.333}{2} = 101.667$$

---

**③ SS_W (집단 내 제곱합) 계산**

$$SS_W = \sum\sum(x_{ij} - \bar{X}_i)^2$$

**비료 A** ($\bar{X}_A = 21.0$):
$(20-21)^2 + (22-21)^2 + (19-21)^2 + (21-21)^2 + (23-21)^2 = 1+1+4+0+4 = 10$

**비료 B** ($\bar{X}_B = 30.0$):
$(30-30)^2 + (28-30)^2 + (32-30)^2 + (29-30)^2 + (31-30)^2 = 0+4+4+1+1 = 10$

**비료 C** ($\bar{X}_C = 25.0$):
$(25-25)^2 + (27-25)^2 + (24-25)^2 + (26-25)^2 + (23-25)^2 = 0+4+1+1+4 = 10$

$$SS_W = 10 + 10 + 10 = \mathbf{30}$$

자유도: $df_W = N - k = 15 - 3 = 12$

$$MS_W = \frac{30}{12} = 2.500$$

---

**④ SS_T 검증**

$$SS_T = \sum\sum x_{ij}^2 - \frac{\left(\sum\sum x_{ij}\right)^2}{N} = 9860 - \frac{380^2}{15} = 9860 - \frac{144400}{15} = 9860 - 9626.667 = 233.333$$

검증: $SS_B + SS_W = 203.333 + 30 = 233.333 = SS_T$ ✓

---

**⑤ F 통계량 계산**

$$F = \frac{MS_B}{MS_W} = \frac{101.667}{2.500} = \mathbf{40.667}$$

---

**⑥ ANOVA 테이블 정리**

| 변동 원인 | SS | df | MS | F | p-value |
|-----------|-----|----|----|---|---------|
| 집단 간 (Between) | 203.333 | 2 | 101.667 | **40.667** | **< 0.001** |
| 집단 내 (Within/Error) | 30.000 | 12 | 2.500 | | |
| **합계 (Total)** | **233.333** | **14** | | | |

---

**⑦ 기각역 확인**

$$F_{0.05,\;2,\;12} = 3.885$$

$$F = 40.667 \gg 3.885 \implies H_0 \text{ 기각}$$

**→ 결론:** 유의수준 5%에서 세 비료 간 평균 수확량은 **유의한 차이가 있다** (F = 40.667, p < 0.001).

---

## 3-3. 사후 검정 (Post-hoc Test) — Tukey HSD

ANOVA에서 $H_0$ 기각 시, 어느 집단 쌍이 다른지 파악

$$HSD = q_{\alpha,k,N-k} \cdot \sqrt{\frac{MS_W}{n}}$$

$$q_{0.05,\;3,\;12} = 3.773 \quad \text{(Studentized Range Table)}$$

$$HSD = 3.773 \times \sqrt{\frac{2.500}{5}} = 3.773 \times 0.707 = 2.667$$

**집단 쌍별 평균 차이:**

| 비교 쌍 | $|\bar{X}_i - \bar{X}_j|$ | > HSD(2.667)? | 결론 |
|---------|--------------------------|---------------|------|
| A vs B | $|21.0 - 30.0| = 9.0$ | ✓ | **유의한 차이** |
| A vs C | $|21.0 - 25.0| = 4.0$ | ✓ | **유의한 차이** |
| B vs C | $|30.0 - 25.0| = 5.0$ | ✓ | **유의한 차이** |

→ **세 비료 모두 서로 유의하게 다름**: B > C > A

# PART 4. 카이제곱 검정 (χ² Test)

## 4-1. 개념 및 종류

카이제곱 검정은 **범주형 데이터**에 적용하며 세 가지 유형이 있다.

| 유형 | 목적 | 귀무가설 |
|------|------|----------|
| **적합도 검정** | 관측 분포가 기대 분포와 같은지 | 주어진 분포를 따른다 |
| **독립성 검정** | 두 범주형 변수가 독립인지 | 두 변수는 독립이다 |
| **동질성 검정** | 여러 집단의 분포가 동일한지 | 집단 간 분포가 같다 |

### 카이제곱 통계량 공식

$$\boxed{\chi^2 = \sum_{i} \frac{(O_i - E_i)^2}{E_i} \sim \chi^2(\nu)}$$

- $O_i$: 관측 빈도 (Observed)
- $E_i$: 기대 빈도 (Expected)
- $\nu$: 자유도

> 기각 조건: $\chi^2 > \chi^2_{\alpha, \nu}$ (항상 우측 단측 검정)

---

## 4-2. 적합도 검정 (Goodness-of-Fit Test)

### 개념
표본이 **이론적 분포**를 따르는지 검정. 자유도 $\nu = k - 1$ ($k$: 범주 수)

### 문제
> 주사위가 공정한지 검정하기 위해 **120번 던진 결과**:

| 눈 | 1 | 2 | 3 | 4 | 5 | 6 | 합계 |
|----|---|---|---|---|---|---|------|
| 관측(O) | 25 | 17 | 22 | 18 | 21 | 17 | 120 |
| 기대(E) | 20 | 20 | 20 | 20 | 20 | 20 | 120 |

α = 0.05

### 풀이

**① 가설 설정**
$$H_0: \text{주사위는 공정하다 (각 눈 확률} = 1/6) \qquad H_1: \text{공정하지 않다}$$

**② 기대 빈도 확인**
$$E_i = n \times p_i = 120 \times \frac{1}{6} = 20 \quad \text{(모든 i에 대해)}$$

> 조건 확인: 모든 $E_i \geq 5$ ✓ (카이제곱 검정 적용 가능)

**③ χ² 통계량 계산**

| 눈 | $O_i$ | $E_i$ | $O_i - E_i$ | $(O_i-E_i)^2$ | $\dfrac{(O_i-E_i)^2}{E_i}$ |
|----|--------|--------|-------------|----------------|--------------------------|
| 1 | 25 | 20 | +5 | 25 | 1.250 |
| 2 | 17 | 20 | -3 | 9 | 0.450 |
| 3 | 22 | 20 | +2 | 4 | 0.200 |
| 4 | 18 | 20 | -2 | 4 | 0.200 |
| 5 | 21 | 20 | +1 | 1 | 0.050 |
| 6 | 17 | 20 | -3 | 9 | 0.450 |
| **합계** | **120** | **120** | **0** | | **2.600** |

$$\chi^2 = 2.600$$

**④ 기각역 확인**

자유도 $\nu = k - 1 = 6 - 1 = 5$:

$$\chi^2_{0.05,\;5} = 11.071$$

$$\chi^2 = 2.600 < 11.071 \implies H_0 \text{ 채택}$$

**⑤ p-값**
$$p\text{-value} \approx 0.761 > 0.05 \implies H_0 \text{ 채택}$$

**→ 결론:** 유의수준 5%에서 주사위가 **공정하지 않다는 증거가 없다** (χ² = 2.600, p = 0.761).

---

## 4-3. 독립성 검정 (Independence Test)

### 개념
두 범주형 변수 간에 **연관성이 있는지** 검정. 분할표(Contingency Table) 사용.

$$E_{ij} = \frac{(\text{행 합계}_i) \times (\text{열 합계}_j)}{N}, \qquad \nu = (r-1)(c-1)$$

### 문제
> 성별과 선호 음료 간 독립성 검정. 200명 설문 결과:

| | 커피 | 녹차 | 주스 | 합계 |
|--|------|------|------|------|
| **남성** | 50 | 30 | 20 | **100** |
| **여성** | 40 | 45 | 15 | **100** |
| **합계** | **90** | **75** | **35** | **200** |

α = 0.05

### 풀이

**① 가설 설정**
$$H_0: \text{성별과 선호 음료는 독립이다} \qquad H_1: \text{독립이 아니다 (연관성 있다)}$$

**② 기대 빈도 계산** $E_{ij} = \dfrac{R_i \times C_j}{N}$

| | 커피 | 녹차 | 주스 |
|--|------|------|------|
| **남성** | $\frac{100\times90}{200}=45.0$ | $\frac{100\times75}{200}=37.5$ | $\frac{100\times35}{200}=17.5$ |
| **여성** | $\frac{100\times90}{200}=45.0$ | $\frac{100\times75}{200}=37.5$ | $\frac{100\times35}{200}=17.5$ |

> 조건 확인: 모든 $E_{ij} \geq 5$ ✓

**③ χ² 통계량 계산**

| 셀 | $O$ | $E$ | $(O-E)^2/E$ |
|----|-----|-----|-------------|
| 남성-커피 | 50 | 45.0 | $(5)^2/45.0 = 0.556$ |
| 남성-녹차 | 30 | 37.5 | $(-7.5)^2/37.5 = 1.500$ |
| 남성-주스 | 20 | 17.5 | $(2.5)^2/17.5 = 0.357$ |
| 여성-커피 | 40 | 45.0 | $(-5)^2/45.0 = 0.556$ |
| 여성-녹차 | 45 | 37.5 | $(7.5)^2/37.5 = 1.500$ |
| 여성-주스 | 15 | 17.5 | $(-2.5)^2/17.5 = 0.357$ |
| **합계** | | | **4.826** |

$$\chi^2 = 4.826$$

**④ 기각역 확인**

자유도 $\nu = (r-1)(c-1) = (2-1)(3-1) = 2$:

$$\chi^2_{0.05,\;2} = 5.991$$

$$\chi^2 = 4.826 < 5.991 \implies H_0 \text{ 채택}$$

**⑤ p-값 및 효과 크기**
$$p\text{-value} \approx 0.090 > 0.05 \implies H_0 \text{ 채택}$$

**Cramér's V** (연관 강도):
$$V = \sqrt{\frac{\chi^2}{N \times \min(r-1, c-1)}} = \sqrt{\frac{4.826}{200 \times 1}} = \sqrt{0.02413} = 0.155$$

**→ 결론:** 유의수준 5%에서 성별과 선호 음료는 **독립적이다** (χ² = 4.826, p = 0.090). 연관 강도는 약함 (V = 0.155).

## 4-4. 동질성 검정 (Homogeneity Test)

### 개념
**여러 집단(모집단)**에서 같은 범주형 변수의 분포가 동일한지 검정.  
독립성 검정과 **계산 방법은 동일**, 차이는 연구 설계에 있음.

### 문제
> 지역(서울/부산/대구) 별로 정치 성향(진보/중도/보수) 분포가 같은지 검정. 300명 조사:

| | 진보 | 중도 | 보수 | 합계 |
|--|------|------|------|------|
| **서울** | 45 | 35 | 20 | **100** |
| **부산** | 25 | 40 | 35 | **100** |
| **대구** | 20 | 30 | 50 | **100** |
| **합계** | **90** | **105** | **105** | **300** |

α = 0.05

### 풀이

**① 가설 설정**
$$H_0: \text{세 지역의 정치 성향 분포가 동일하다} \qquad H_1: \text{적어도 하나의 지역이 다르다}$$

**② 기대 빈도 계산** $E_{ij} = \dfrac{R_i \times C_j}{N}$

| | 진보 | 중도 | 보수 |
|--|------|------|------|
| **서울** | $\frac{100\times90}{300}=30$ | $\frac{100\times105}{300}=35$ | $\frac{100\times105}{300}=35$ |
| **부산** | 30 | 35 | 35 |
| **대구** | 30 | 35 | 35 |

**③ χ² 통계량 계산**

| 셀 | $O$ | $E$ | $(O-E)^2/E$ |
|----|-----|-----|-------------|
| 서울-진보 | 45 | 30 | $225/30 = 7.500$ |
| 서울-중도 | 35 | 35 | $0/35 = 0.000$ |
| 서울-보수 | 20 | 35 | $225/35 = 6.429$ |
| 부산-진보 | 25 | 30 | $25/30 = 0.833$ |
| 부산-중도 | 40 | 35 | $25/35 = 0.714$ |
| 부산-보수 | 35 | 35 | $0/35 = 0.000$ |
| 대구-진보 | 20 | 30 | $100/30 = 3.333$ |
| 대구-중도 | 30 | 35 | $25/35 = 0.714$ |
| 대구-보수 | 50 | 35 | $225/35 = 6.429$ |
| **합계** | | | **25.952** |

$$\chi^2 = 25.952$$

**④ 기각역 확인**

자유도 $\nu = (3-1)(3-1) = 4$:

$$\chi^2_{0.05,\;4} = 9.488$$

$$\chi^2 = 25.952 \gg 9.488 \implies H_0 \text{ 기각}$$

**⑤ p-값**
$$p\text{-value} \approx 0.00003 < 0.05 \implies H_0 \text{ 기각}$$

**→ 결론:** 유의수준 5%에서 세 지역의 정치 성향 분포는 **유의하게 다르다** (χ² = 25.952, p < 0.001).  
서울은 진보 성향이, 대구는 보수 성향이 상대적으로 강하게 나타남.

## 4-5. 카이제곱 검정 주의사항

| 조건 | 내용 |
|------|------|
| 기대 빈도 | 모든 셀의 $E_i \geq 5$ 필요 (위반 시 Fisher's Exact Test 사용) |
| 표본 크기 | 충분히 커야 함 ($N \geq 20$ 권장) |
| 독립성 | 각 관측치는 서로 독립 |
| 방향성 없음 | 항상 **우측 단측 검정** (χ² ≥ 0) |
| 효과 크기 | Cramér's V (2×2: Phi 계수) 로 측정 |

# PART 5. 전체 비교 요약

## 검정법 선택 기준

```
데이터 유형
  │
  ├─ 연속형 ─── 집단 수
  │               ├─ 1개 ──── 모분산 앎? ─── Yes → Z검정
  │               │                      └─ No  → 단일 t검정
  │               ├─ 2개 ──── 모분산 앎? ─── Yes → Z검정
  │               │                      └─ No  → 등분산F검정 → 합동/Welch t검정
  │               └─ 3개+ ──────────────────────→ ANOVA → 사후검정
  │
  └─ 범주형 ─── 목적
                  ├─ 하나의 분포 vs 이론 → 적합도 χ²검정
                  ├─ 두 변수 간 연관성  → 독립성 χ²검정
                  └─ 여러 집단 분포 비교 → 동질성 χ²검정
```

## 네 검정법 핵심 비교표

| 구분 | t검정 | F검정 (등분산) | ANOVA | χ²검정 |
|------|-------|--------------|-------|--------|
| 데이터 | 연속형 | 연속형 | 연속형 | **범주형** |
| 목적 | 평균 비교 | 분산 비교 | 3집단+ 평균 | 분포/독립성 |
| 귀무가설 | $\mu_1 = \mu_2$ | $\sigma_1^2 = \sigma_2^2$ | $\mu_1 = \cdots = \mu_k$ | 독립/적합 |
| 통계량 | $t = \dfrac{\bar{X}-\mu}{S/\sqrt{n}}$ | $F = \dfrac{S_1^2}{S_2^2}$ | $F = \dfrac{MS_B}{MS_W}$ | $\chi^2 = \sum\dfrac{(O-E)^2}{E}$ |
| 분포 | t분포 ($n-1$) | F분포 ($\nu_1,\nu_2$) | F분포 ($k-1,N-k$) | χ²분포 ($\nu$) |
| 검정 방향 | 양측/단측 | 양측/단측 | 우측 단측 | **우측 단측** |
| 사후검정 | 불필요 | 불필요 | Tukey, Scheffe | 잔차 분석 |
